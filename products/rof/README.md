# Retail Opportunity Finder

Retail Opportunity Finder (ROF) is organized into layers:

- `analytics/`: notebooks, dashboards, and anlytics scripts
- `data_platforn`: SQL and R files that build our database layers 
- `docs`: information about the repo
- `outputs`: outputs from analyses
- `qa`: scripts for monitoring data quality
