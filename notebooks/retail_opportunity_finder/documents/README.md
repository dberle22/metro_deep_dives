# ROF Documents Hub

Central location for non-code project documents.

## Structure

- `plans/`: build plans, flow definitions, implementation plans
- `sprints/`: sprint overviews, checklists, integration sprint tracking
- `improvements/`: MVP and V2 improvement backlogs and decision logs

## Notes

- Section code and artifact contracts remain in `sections/`
- Integration/render files remain in `integration/qmd/`
- Public web payload remains under repository `docs/rof-mvp/`
