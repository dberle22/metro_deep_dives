# Retail Opportunity Finder MVP (Published Site)

This folder is the GitHub Pages payload for the ROF MVP report.

## Update workflow

1. Re-render and publish into this folder:

```bash
./scripts/publish_rof_mvp.sh
```

2. Commit and push changes under `docs/rof-mvp`.

## GitHub Pages settings

- Branch: `main`
- Folder: `/docs`

The public URL will be:

- `https://<org-or-user>.github.io/<repo>/rof-mvp/`
